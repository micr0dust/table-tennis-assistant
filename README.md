# 桌球雙打輔助程式

打了桌球雙打後覺得計分好難，還要記什麼時候誰發球。於是我寫了這個桌球雙打輔助程式，會幫忙計分和計誰發球，看有沒有人路過要用的。因為要說幾比幾和提醒我們這些桌球新手誰要發球實在太累了，所以我還加了語音功能，自動說比分和哪邊發球，免得浪費唇舌。

一開始點的那邊表示是先發球的一方，之後就照規則繼續。
有 Bug 或我規則搞錯的話可以反映。
